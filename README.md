## 🚀 Features

- **Infinite Masonry Gallery:** Explore thousands of high-res images with a sleek, masonry-style layout.
- **Smart Infinite Scroll:** Assets load automatically as you scroll, keeping your workflow uninterrupted.
- **Dynamic Resizing:** Double-click any image to open a professional resize modal—get the exact dimensions you need.
- **Dev-Friendly Copying:** Quickly copy raw CDN URLs or ready-to-use HTML `<img />` tags.
- **VS Code Native UI:** Designed to feel like a built-in part of your editor with theme-aware styling.

---

## 🛠️ Quick Start

1. **Open the Activity Bar:** Click the new **Unsplash Pro** icon on your left sidebar.
2. **Search:** Type any keyword (e.g., `workspace`, `minimalist`, `nature`) in the search bar.
3. **Copy:** - **Hover** over any image to see quick copy actions.
   - **Double-click** to open the resize tool and get custom dimensions.
4. **Integrate:** Paste directly into your project files.
